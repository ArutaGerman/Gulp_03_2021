Актуальная сборка Gulp от марта 2021г.

Сборка с использованием BEM.

Точки для адаптивного дизайна находятся в файле \src\scss\include и называются $breakpoints

Сборка картинок и svg спрайтов производится отдельными тасками.

Production mode:
gulp

Собрать SvgSprites:
gulp svgSprite

Build mode: 
gulp build
